# 工作区模板

工作区模板是预设的工作区结构，包含文件布局、环境变量和初始配置。创建新工作区时选择模板，一键完成初始化。

## 内置模板

系统预置四种模板：

| 模板 | 说明 |
|------|------|
| 空白工作区 | 只有基础的 `.aiasys/` 目录结构，无预设内容 |
| 数据分析 | 预设 data/、notebooks/、outputs/ 目录，附带 pandas/matplotlib 等依赖声明 |
| 文献综述 | 预设 papers/、notes/、references/ 目录，附带 arXiv 检索相关配置 |
| 项目开发 | 预设 src/、tests/、docs/ 目录，附带 Python 项目骨架 |

## 创建工作区时选择模板

创建新工作区时，模板选择器展示所有可用模板。选择模板后系统自动创建目录结构、写入预设文件和初始环境变量。

## 模板预览

模板选择器中每个模板可以展开查看文件树结构。预览以树形目录展示模板包含的全部文件和子目录，方便用户判断模板是否符合需求。

## 推荐能力

每个模板可以关联推荐的 MCP 和 Skill。创建完成后，系统提示用户安装推荐的能力扩展。用户可以选择安装或跳过。

## 保存当前工作区为模板

将当前工作区的内容保存为新模板，供后续复用。保存时可以选择：

- 包含哪些文件和目录
- 是否包含环境变量
- 是否包含已安装的 MCP 和 Skill 列表

保存后的模板出现在模板列表中，可以在创建新工作区时选择。

## 模板管理

模板管理界面支持：

- 查看模板列表和详细信息
- 导出模板为 zip 文件
- 删除不再需要的模板

内置模板不可删除。

## 外部模板导入

支持从外部导入模板。导入方式：

- 上传 zip 格式的模板包
- 从外部模板市场导入

导入的模板经过校验后加入本地模板列表，可以在创建新工作区时使用。
